Set up CS accounts

  1. Log in to PipelineMT, click class Email List
  2. Copy the Student Email List
  3. Log in to https://mgt.cs.mtsu.edu/aru
  4. Paste the Student Email List
  5. Assign CS accounts and password to student list

handin Setup

  1. Log in to 00 account: ln -s /nfshome/untch/bin
  2. Create a folder named 2170
  3. move class roaster 2170.csv to the folder 2170
  4. cd to 2170 folder: gusinit 2170
  5. cd to xyang/public_html/2170 folder: change the Notes.txt file
  6. Method 1 (work for one section):
    • move HMWK.csv to the folder 2170
    • handin ".HMWK" HMWK.csv
  7. Method 2 (work for several sections):
    • move H.desc and H.due to the folder 2170
    • gushmwk 2170 001
    • gushmwk 2170 003
  1. Errors: Directory not that deep (Solution: svn delete)
