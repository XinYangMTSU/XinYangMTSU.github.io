You are working for a IT Company and your boss wants you to develop a web-based scientific calculator, where the user can enter a mathematical expression using clickable buttons for the digits 0-9, decimal point, and operators (+, -, *, /). When the user clicks the "=" button, the form will be submitted using the POST method to the php script part, and the result will be displayed. Please add fancy style to the calculator. Please finish all steps by following instructions.
