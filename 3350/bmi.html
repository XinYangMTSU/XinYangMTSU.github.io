Interaction Features – Combining Existing Variables

Interaction features are new variables created by combining two or more existing features in a way that captures their relationship. For example, BMI is an interaction between weight and height, and Age-adjusted BMI captures how health risks may change with both body size and age.

Example: Creating BMI and Age-Adjusted BMI

  
import pandas as pd

# Example dataset: patient records
data = {
    "patient_id": [1, 2, 3, 4],
    "weight_kg": [70, 80, 60, 90],   # in kilograms
    "height_m": [1.75, 1.80, 1.60, 1.95],  # in meters
    "age": [25, 40, 35, 50]
}

df = pd.DataFrame(data)
print("Original Data:")
print(df)

# --- Create Interaction Features ---
# 1. BMI (interaction between weight and height)
df["BMI"] = df["weight_kg"] / (df["height_m"] ** 2)

# 2. Age-adjusted BMI (interaction between BMI and age)
df["BMI_age_adjusted"] = df["BMI"] * df["age"]

print("\nData with Interaction Features:")
print(df)

Notes:

  • BMI = weight ÷ height², a health indicator.
  • BMI_age_adjusted = BMI × age, highlighting combined effects of age and BMI.
  • Interaction features often reveal patterns that single variables cannot capture on their own.
