Fixing Inconsistent Formats in Python

When cleaning real-world datasets, you often find inconsistent formats. For example:

  • "N/A", "na", "null" → should all be standardized to NaN.
  • "Male", "M", "male" → should all be unified to the same category.
  
import pandas as pd
import numpy as np

# Create messy dataset
data = {'Name': ['Alice', 'Bob', 'Charlie', 'David'],
        'Gender': ['Male', 'M', 'male', 'F'],
        'Score': [85, 'N/A', 'na', 'null']}

df = pd.DataFrame(data)
print("Original Data:")
print(df)

# Step 1: Standardize missing values
df['Score'] = df['Score'].replace(['N/A', 'na', 'null'], np.nan)

# Step 2: Unify categorical values
df['Gender'] = df['Gender'].replace({'M': 'Male', 'male': 'Male', 'F': 'Female'})

print("\nCleaned Data:")
print(df)
