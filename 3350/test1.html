  • ★ Chapter 3: Supervised Learning: Regression
    1. * Linear Regression
      • Objective Function

        • \( \min_{\boldsymbol{\beta}} \|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|_2^2 \)

    2. * Polnomial Regression
      • Feature Expansion
    3. * Ridge Regression
      • Objective Function

        • \( \min_{\boldsymbol{\beta}} \|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|_2^2 + \lambda \|\boldsymbol{\beta}\|_2^2 \)

      • L2 penalty
    4. * Lasso Regression
      • Objective Function

        • \( \min_{\boldsymbol{\beta}} \|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|_2^2 + \lambda \|\boldsymbol{\beta}\|_1 \)

      • L1 penalty
    5. * Elastic Net Regression
      • Objective Function

        • \( \min_{\boldsymbol{\beta}} \|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|_2^2 + \lambda_1 \big( \alpha \|\boldsymbol{\beta}\|_1 + \lambda_2 \|\boldsymbol{\beta}\|_2^2 \big) \)

      • L1 and L2 penalties
    6. * Model Evaluation
      • SSE
      • MSE
      • RMSE
      • R^2
    7. * Overfitting vs Underfitting
