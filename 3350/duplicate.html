Removing Duplicates in Python

  
    # Imports the pandas library and gives it the shorthand name "pd"
    import pandas as pd
    
    # Create simple dataset with duplicates
    
    # Creates a dictionary with two lists: names and ages
    data = {'Name': ['Alice', 'Bob', 'Alice', 'Charlie'], 
            'Age': [25, 30, 25, 35]}
    
    # pd.DataFrame(data) converts the dictionary into a pandas DataFrame (like a table)
    df = pd.DataFrame(data)
    
    print("Original data before dropping duplicates:")
    print(df)
    print(f"\nRows before dropping duplicates: {len(df)}")
    
    # Check if ANY duplicates exist
    has_duplicates = df.duplicated().any()
    print(f"\nHas duplicates: {has_duplicates}")  # True
    
    # Remove duplicates
    df_clean = df.drop_duplicates()
    print(f"\nRows after dropping duplicates: {len(df_clean)}")
    print("\nCleaned data after dropping duplicates::")
    print(df_clean)

What is a Dictionary?

A dictionary in Python is a data structure that stores data in key-value pairs. Think of it like a real dictionary where you look up a word (key) to find its definition (value).

Dictionary Basics

Creating:

my_dict = {'key1': 'value1', 'key2': 'value2'}

Accessing values:

names = data['Name']  # Gets ['Alice', 'Bob', 'Alice', 'Charlie']
    ages = data['Age']    # Gets [25, 30, 25, 35]

Why use dictionaries for DataFrames?

  • Each key becomes a column name
  • Each value (list) becomes the column data
  • pandas uses this structure to create organized tables

What does f"" mean?

The f"" is called an f-string (formatted string literal) in Python. It's a way to embed variables and expressions directly inside strings.

How it works:

Without f-string (old way):

name = "Alice"
    age = 25
    print("Hello, my name is " + name + " and I am " + str(age) + " years old")

With f-string (modern way):

name = "Alice"
    age = 25
    print(f"Hello, my name is {name} and I am {age} years old")
