CSCI 3350: Machine Learning Final Project

Guidelines & Requirements

Project Requirements

Your final project must include all the following:
  • ★ Dataset Selection: Any dataset (Kaggle, UCI, Google Dataset Search, etc.)
  • ★ Data Preprocessing & EDA: Cleaning, handling missing data, visualization (matplotlib)
  • ★ Algorithms: Use and compare at least 3 ML models
  • ★ Cross-Validation: Use 5-fold or 10-fold CV for all models
  • ★ Evaluation Metrics:
    • ☆ Classification: Accuracy, Precision, Recall, F1, Confusion Matrix, ROC/AUC
    • ☆ Regression: MSE, RMSE, MAE, R²
    • ☆ Comparison table for all models
  • ★ Code: Python with scikit-learn, pandas, numpy, matplotlib
  • ★ Visualizations: Include 5–7 charts (plots, heatmap, ROC, etc.)
  • ★ Final Report (PDF): 8–12 pages
    • ☆ Introduction
    • ☆ Methodology
    • ☆ Experiments & Results
    • ☆ Discussion
    • ☆ Conclusion
  • ★ Presentation: 10 minutes per student

Approved Algorithms

Classification: Logistic Regression, Decision Trees, Random Forest, SVM, KNN, Naive Bayes, Gradient Boosting

Regression: Linear Regression, Ridge/Lasso, SVR, Random Forest Regressor, Gradient Boosting

Submission Checklist

  • ★ Jupyter Notebook (.ipynb) or Python script (.py)
  • ★ Dataset (or link to download)
  • ★ README instructions
  • ★ Final Report (PDF)
