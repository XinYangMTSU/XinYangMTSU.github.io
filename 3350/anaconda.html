How to Install Anaconda and Set Up Jupyter Notebook

This guide walks you through installing the Anaconda Distribution and launching Jupyter Notebook for your Machine Learning coursework.

Step-by-Step Instructions

Step 1: Download Anaconda

  1. Go to the official Anaconda website:
    https://www.anaconda.com/products/distribution
  2. Click the "Skip Registration" to skip registration.
  3. Click the "Download" button for your operating system (Windows, macOS, or Linux).
  4. Choose the Python 3.x version (e.g., Python 3.11). Do NOT choose the "Miniconda" version.

Step 2: Install Anaconda

  1. Open the downloaded installer file and follow the on-screen instructions.
  2. For Windows: choose to add Anaconda to your PATH (optional but useful).
  3. Accept default settings unless you know what you're doing.

Step 3: Launch Anaconda Navigator

  1. Once installed, open Anaconda Navigator from your Start Menu (Windows) or Applications (Mac).
  2. From Anaconda Navigator, click "Launch" next to Jupyter Notebook.
  3. A browser window will open showing the Jupyter dashboard.

Step 4: Open or Create a Notebook

  1. In the Jupyter browser interface, navigate to a folder where you want to work.
  2. Click on New â†’ Python 3 (ipykernel) to create a new notebook.
  3. Begin typing and running Python code in the cells.

Step 5: (Optional) Install Additional Libraries

To install libraries like scikit-learn, matplotlib, or pandas, open Anaconda Prompt or Terminal and type:

conda install scikit-learn matplotlib pandas

Troubleshooting

  • If Jupyter Notebook does not launch, try launching it via Anaconda Prompt: 
    jupyter notebook
  • Make sure your browser does not block pop-ups.
  • If Anaconda Navigator is slow, you can use the command line to manage environments and launch notebooks.
