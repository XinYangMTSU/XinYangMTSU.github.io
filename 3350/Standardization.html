Standardization (Z-score): mean = 0, variance = 1

Standardization (also called Z-score normalization) transforms data so that it has:

  • Mean = 0
  • Variance = 1 (Standard Deviation = 1)

Formula for standardization:

\( z = \frac{(x - μ)}{σ} \)

Where:

  • x = original value
  • μ = mean of the feature
  • σ = standard deviation of the feature

Python Example: Standardization with scikit-learn

  
import pandas as pd
from sklearn.preprocessing import StandardScaler

# Sample dataset
data = {'Score': [50, 80, 90, 100, 60]}
df = pd.DataFrame(data)

print("Original Data:")
print(df)

# Initialize scaler
scaler = StandardScaler()

# Fit and transform
df['Score_Standardized'] = scaler.fit_transform(df[['Score']])

print("\nAfter Standardization (Z-score):")
print(df)

Why Standardize?

  • Many ML algorithms (e.g., Logistic Regression, KNN, Neural Networks) perform better when features are on a similar scale.
  • Helps gradient-based methods converge faster.
  • Prevents features with large values (like income = thousands, age = tens) from dominating the model.
  • Ensures features with different scales (like height in cm and weight in kg) contribute fairly.
