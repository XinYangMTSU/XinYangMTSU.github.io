1) What is Pandas?

Pandas is a fast, flexible Python library for data analysis and manipulation. It provides labeled, table‑like data structures and rich utilities to load, clean, reshape, and analyze data. Think “Excel for Python,” but more powerful and scalable.

  • Built on NumPy, integrates with Matplotlib and Scikit‑learn.
  • Works with CSV, Excel, SQL, JSON, Parquet, and more.
  • Great for cleaning (missing values, duplicates), transformation, grouping, joining, and time series.

3) Core Data Structures

  • Series – 1D labeled array (like a single column).
  • DataFrame – 2D labeled table (like a sheet).
import pandas as pd

s = pd.Series([10, 20, 30], index=["a","b","c"])   # Series
print(s)

df = pd.DataFrame({"Name":["Alice","Bob"], "Age":[25,30]})  # DataFrame
print(df)

5) Inspect & Summarize

print(df.head())        # first rows
print(df.info())        # column types + non-null counts
print(df.describe())    # stats for numeric
print(df.isna().mean()) # missing-value ratio per column
print(df["dept"].value_counts())
