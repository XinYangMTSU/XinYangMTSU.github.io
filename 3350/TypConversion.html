Type Conversion – Ensure Columns Have the Right Data Type

In real datasets, some columns may be stored as the wrong type, such as numbers stored as text or dates stored as plain strings. Type conversion makes sure each column has the correct type (e.g., int, float, datetime).

Example: Converting Strings to Numbers and Dates

  
import pandas as pd

# Sample dataset with mixed types
data = {
    'Price': ['100', '250', '300', '400'],
    'Date': ['2023-01-05', '2023-02-10', '2023-03-15', '2023-04-20']
}

df = pd.DataFrame(data)
print("Original Data Types:")
print(df.dtypes)

# Convert 'Price' column from string to integer
df['Price'] = df['Price'].astype(int)

# Convert 'Date' column from string to datetime
df['Date'] = pd.to_datetime(df['Date'])

print("\nAfter Conversion:")
print(df.dtypes)
print(df)

Notes:

  • astype() is used for numeric conversions (e.g., string → int/float).
  • pd.to_datetime() is used for converting text into datetime objects.
  • Correct data types improve performance and allow accurate analysis.
