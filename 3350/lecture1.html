Introduction to Machine Learning

1. What is Machine Learning?

Machine learning (ML) is the process of training a model—a software component—to make useful predictions or generate content from data :contentReference[oaicite:2]{index=2}.

2. Types of ML Systems

ML systems include:

  • Supervised: trained with labeled data
  • Unsupervised: finding structure in unlabeled data
  • Reinforcement: learning via environment feedback

4. Regression

Predict continuous values (e.g., house prices, temperature).

# Example (scikit‑learn)
from sklearn.linear_model import LinearRegression
model = LinearRegression()
model.fit(X_train, y_train)

5. Classification

Assign inputs to discrete categories (e.g., spam detection, digit recognition).

# Example (scikit‑learn)
from sklearn.tree import DecisionTreeClassifier
clf = DecisionTreeClassifier()
clf.fit(X_train, y_train)

6. Unsupervised Learning

Discover patterns without labeled outputs—like clustering, dimensionality reduction.

7. Reinforcement Learning

Agents learn by interacting with environments and receiving feedback (rewards or penalties).

8. Generative AI

Models that generate new data—text, images, audio—based on learned patterns (e.g., GPT, GANs).
