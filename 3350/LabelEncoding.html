Label Encoding: Assigns an integer to each category

Label Encoding is a technique that converts categorical values into integers. Each category gets a unique integer label: 


Red   → 0
Blue  → 1
Green → 2

⚠️ Caution: This can mislead machine learning models if the categories are nominal (no natural order). The model may wrongly interpret Green (2) > Blue (1) > Red (0) as meaningful.

Python Example: Label Encoding with scikit-learn

  
import pandas as pd
from sklearn.preprocessing import LabelEncoder

# Sample dataset
data = {'Size': ['Small', 'Medium', 'Large', 'Medium', 'Small']}
df = pd.DataFrame(data)

print("Original Data:")
print(df)

# Apply Label Encoding
encoder = LabelEncoder()
df['Size_Encoded'] = encoder.fit_transform(df['Size'])

print("\nAfter Label Encoding:")
print(df)

When to Use Label Encoding?

  • Works well when categories are ordinal (e.g., Small < Medium < Large).
  • Simple and efficient when categories have a meaningful order.

Limitations

  • Not suitable for nominal (unordered) categories like colors, countries, or names.
  • Models may misinterpret the encoded integers as ordered or having magnitude.
  • For unordered categories, One-Hot Encoding is usually safer.
