Anaconda — Quickstart & Cheatsheet

1) What is Anaconda?

Anaconda is a free, open-source distribution of Python (and R) designed for data science, machine learning, and scientific computing. Think of it as a package manager + environment manager + big toolbox in one installer.

  • Ships with hundreds of libraries (NumPy, Pandas, Matplotlib, Scikit‑learn, Jupyter).
  • Includes conda for package and environment management.
  • Works on Windows, macOS, and Linux.

2) Why use it?

  • Faster setup: everything you need for class and research is prebundled.
  • Fewer conflicts: isolate projects so versions don’t clash.
  • Beginner‑friendly: Anaconda Navigator provides a GUI to launch tools and manage environments.

3) Install & Setup

Download Anaconda (Python 3.11) from the official site for your OS, then follow the steps below.

  • Official download page

Windows & macOS

  • Run the installer, choose "Just Me" (recommended) and let it add Anaconda to your PATH if prompted.
  • Open the Anaconda Prompt (Windows) or Terminal (macOS) to use conda.

Linux

# Example: install to $HOME/anaconda3
bash Anaconda3-*-Linux-x86_64.sh
# follow prompts, then restart your shell

Verify

conda --version
python -V
python -c "import numpy, pandas, sklearn; print('OK')"

4) Conda Basics

# Search and install a package
conda search pandas
conda install pandas

# Update conda & packages
conda update conda
conda update --all

# Remove a package
conda remove package-name

5) Environments

Use separate environments to keep project dependencies clean and reproducible.

Create / Activate / List

# Create a new environment (Python 3.11)
conda create -n ml-env python=3.11 -y
conda activate ml-env
conda env list

Install Common Tools

conda install numpy pandas matplotlib scikit-learn jupyter -y

Remove

conda deactivate
conda env remove -n ml-env

6) Jupyter Notebook

# Install (if not already)
conda install jupyter -y

# Launch in current environment
jupyter notebook
# or
jupyter lab

Tip: Create a dedicated environment per course or project and launch Jupyter from within that environment.

7) Anaconda Navigator (GUI)

  • Open Anaconda Navigator from your applications menu.
  • Use the left sidebar to select an environment.
  • Launch Jupyter Notebook or Spyder.
  • Install/update packages via the Environments tab.

8) Troubleshooting

Fix SSL / network issues

conda config --set ssl_verify true
conda clean --all -y

Speed up installs (optional)

# Mamba is a fast drop-in replacement for conda
conda install -n base -c conda-forge mamba
mamba install pandas numpy

PATH or activation issues

# Initialize conda for your shell
conda init
# then restart your terminal
