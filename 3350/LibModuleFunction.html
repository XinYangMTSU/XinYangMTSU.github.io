Quick Guide: sklearn library module function sklearn.datasets matplotlib

1) Overview (The Big Picture)

In Python, code is organized hierarchically: 


# Library (a collection of modules/packages)
import sklearn            # the scikit-learn machine learning library

# Module (a file or subpackage with code you can import)
from sklearn import datasets   # a module inside sklearn

# Function (a specific tool inside a module)
from sklearn.datasets import load_iris  # one function from that module

iris = load_iris()  # use it

2) Definitions

Library

  • A library is a collection of reusable code (modules/packages) you can import to avoid reinventing the wheel.
  • Examples: numpy, pandas, matplotlib, sklearn.

Module

  • A module is a single Python file (something.py) containing functions, classes, or variables.

Function (and Class)

  • A function is a named block of code you can call (e.g., load_iris()).
  • A class defines objects/behaviors (e.g., LinearRegression), which you instantiate then .fit()/.predict().

3) Import Patterns

Import a module


from sklearn import datasets
iris = datasets.load_iris()

Import a specific function or class


from sklearn.datasets import load_iris
iris = load_iris()

4) What is sklearn.datasets?

sklearn.datasets is a module that provides toy datasets, fetchers for real datasets, and synthetic data generators.

  • Toy loaders: load_iris(), load_diabetes(), load_wine(), load_breast_cancer()
  • Fetchers: fetch_california_housing(), fetch_20newsgroups(), fetch_openml()

Example (Diabetes as DataFrame)


from sklearn.datasets import load_diabetes

# Return a Bunch with a pandas DataFrame
D = load_diabetes(as_frame=True)
df = D.frame  # features + target
print(df.head())

# Access pieces
X = D.data
y = D.target
print(D.feature_names)  # ['age','sex','bmi','bp','s1','s2','s3','s4','s5','s6']

5) Discoverability: dir() & help()

Use Python’s built‑ins to explore what’s inside a library/module/object and how to use it.

dir() — list available names


import sklearn
for name in dir(sklearn):
    print(name)

help() — show docstring/usage


from sklearn.datasets import load_iris
help(load_iris)      # open docs in the console

6) FAQ

Is Matplotlib part of sklearn?

  • No. It’s a separate plotting library: import matplotlib.pyplot as plt.

Quick plotting starter


import matplotlib.pyplot as plt
from sklearn.datasets import load_diabetes

# Load the diabetes dataset as a pandas DataFrame (instead of raw NumPy arrays)
D = load_diabetes(as_frame=True)
df = D.frame

# Create a scatter plot
plt.scatter(df["bmi"], df["target"], alpha=0.6)
plt.title("BMI vs Diabetes Progression")
plt.xlabel("BMI (standardized)")
plt.ylabel("Progression score (1 yr)")
plt.show()
