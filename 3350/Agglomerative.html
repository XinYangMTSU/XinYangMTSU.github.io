Agglomerative Clustering (Bottom-Up)

Step-by-step agglomerative clustering showing dendrogram construction for 5 points

Problem Statement

Given:

  • Dataset: 5 data points: P1(2,3), P2(2,5), P3(5,3), P4(6,7), P5(8,6)
  • Linkage method: Complete linkage (maximum distance)
  • Distance metric: Euclidean distance
  • Goal: Build complete dendrogram and identify 2 final clusters

Step 0: Initialization

Initial state: Each point is its own cluster

Initial Distance Matrix Calculation

Distance formula: \(d = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}\)

All Pairwise Distance Calculations:

P1(2,3) to P2(2,5): $d = \sqrt{(2-2)^2 + (3-5)^2} = \sqrt{0 + 4} = \mathbf{2.00}$ P1(2,3) to P3(5,3): $d = \sqrt{(2-5)^2 + (3-3)^2} = \sqrt{9 + 0} = \mathbf{3.00}$ P1(2,3) to P4(6,7): $d = \sqrt{(2-6)^2 + (3-7)^2} = \sqrt{16 + 16} = \mathbf{5.66}$ P1(2,3) to P5(8,6): $d = \sqrt{(2-8)^2 + (3-6)^2} = \sqrt{36 + 9} = \mathbf{6.71}$
P2(2,5) to P3(5,3): $d = \sqrt{(2-5)^2 + (5-3)^2} = \sqrt{9 + 4} = \mathbf{3.61}$ P2(2,5) to P4(6,7): $d = \sqrt{(2-6)^2 + (5-7)^2} = \sqrt{16 + 4} = \mathbf{4.47}$ P2(2,5) to P5(8,6): $d = \sqrt{(2-8)^2 + (5-6)^2} = \sqrt{36 + 1} = \mathbf{6.08}$
P3(5,3) to P4(6,7): $d = \sqrt{(5-6)^2 + (3-7)^2} = \sqrt{1 + 16} = \mathbf{4.12}$ P3(5,3) to P5(8,6): $d = \sqrt{(5-8)^2 + (3-6)^2} = \sqrt{9 + 9} = \mathbf{4.24}$
P4(6,7) to P5(8,6): $d = \sqrt{(6-8)^2 + (7-6)^2} = \sqrt{4 + 1} = \mathbf{2.24}$

Key Insights from Agglomerative Clustering

  • Bottom-Up Process: Starts with individual points as separate clusters and iteratively merges the most similar pairs until a single cluster remains.
  • Dendrogram Representation: A tree-like diagram (dendrogram) visually shows the merging sequence and the distance at which clusters combine.
  • Flexibility in Cluster Count: You can choose the desired number of clusters by “cutting” the dendrogram at a chosen height.
  • Choice of Linkage Matters: Different linkage criteria—such as single, complete, or average—affect the shape and compactness of clusters.
  • No need to specify k upfront: The dendrogram shows clustering structure at all levels
  • Clear hierarchical relationships: P1 and P2 are most similar (merge at 2.00), followed by P4 and P5 (merge at 2.24)
  • Natural clustering: Large gap between heights 3.61 and 6.71 suggests 2 clusters is optimal
