Correlation Heatmap – Visualizing Feature Relationships

A heatmap provides a color-coded visualization of the correlation matrix, making it easy to spot which features are strongly related (positively or negatively). This is especially useful when you have many features in your dataset.

Example: Correlation Heatmap with Seaborn

  
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Example dataset: student performance
data = {
    "study_hours": [5, 10, 2, 8, 12, 3],
    "sleep_hours": [7, 6, 8, 5, 6, 9],
    "exam_score": [75, 88, 50, 82, 95, 55]
}

df = pd.DataFrame(data)

# --- Calculate correlation matrix ---
corr_matrix = df.corr()
print("Correlation Matrix:")
print(corr_matrix)

# --- Plot heatmap ---
plt.figure(figsize=(6,4))
sns.heatmap(corr_matrix, annot=True, cmap="coolwarm", center=0, linewidths=0.5)
plt.title("Correlation Heatmap", fontsize=14)
plt.show()

Notes:

  • df.corr() → computes the correlation coefficient (values between -1 and +1) for every pair of numeric features in the dataset.
  • sns.heatmap() → creates a heatmap visualization of the correlation matrix:
    • annot=True shows the correlation values inside the boxes.
    • cmap="coolwarm" colors positive correlations red and negative correlations blue.
    • center=0 ensures that white is at 0 (neutral relationship).
  • Interpretation:
    • Values close to +1 → strong positive correlation (both increase together).
    • Values close to -1 → strong negative correlation (one increases, the other decreases).
    • Values near 0 → no linear relationship.
