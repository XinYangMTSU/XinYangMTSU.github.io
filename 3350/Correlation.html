Correlation scores measure the strength of the linear relationship between a numerical feature and the target variable. A high absolute correlation suggests that the feature may be useful for prediction. However, correlation only captures linear relationships and may miss more complex patterns.

Example: Checking Correlation with Target

  
import pandas as pd

# Example dataset: student performance
data = {
    "study_hours": [5, 10, 2, 8, 12, 3],
    "sleep_hours": [7, 6, 8, 5, 6, 9],
    "exam_score": [75, 88, 50, 82, 95, 55]
}

df = pd.DataFrame(data)
print("Original Data:")
print(df)

# --- Calculate Correlation Scores ---
correlation = df.corr()["exam_score"].sort_values(ascending=False)
print("\nCorrelation with Exam Score:")
print(correlation)

Notes:

  • Correlation values range from -1 (perfect negative) to +1 (perfect positive).
  • Features with high correlation (positive or negative) to the target are often good candidates.
  • Beware of multicollinearity: if two features are highly correlated with each other, it can harm model performance.
