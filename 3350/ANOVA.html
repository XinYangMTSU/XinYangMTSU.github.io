ANOVA F-Test â€“ Testing Feature Variance Across Groups

The ANOVA F-test (Analysis of Variance) is used to measure how well a numerical feature separates different groups of a categorical target. It compares the variance between groups with the variance within groups. A higher F-score means the feature is more useful for distinguishing between classes.

Example: ANOVA F-Test for Feature Selection

  
import pandas as pd
from sklearn.feature_selection import f_classif, SelectKBest

# Example dataset: student exam scores across pass/fail
data = {
    "hours_studied": [2, 5, 8, 10, 12, 3, 6, 9],
    "attendance_rate": [60, 70, 85, 90, 95, 65, 80, 92],
    "passed_exam": [0, 0, 1, 1, 1, 0, 1, 1]
}

df = pd.DataFrame(data)
print("Original Data:")
print(df)

X = df[["hours_studied", "attendance_rate"]]
y = df["passed_exam"]

# --- Apply ANOVA F-Test ---
f_scores, p_values = f_classif(X, y)

for feature, score, p in zip(X.columns, f_scores, p_values):
    print(f"{feature}: F = {score:.2f}, p-value = {p:.4f}")

# Select top k features
X_new = SelectKBest(f_classif, k=1).fit_transform(X, y)

Notes:

  • Use when features are continuous and the target is categorical.
  • A high F-score and low p-value suggest the feature is important for class separation.
  • ANOVA assumes data is normally distributed and groups have similar variance.
