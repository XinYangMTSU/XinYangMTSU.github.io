Lesson 2 • From one vertex to all vertices

Put All Vertices Together: From \(y=X\beta\) to \(Y=XB+E\)

In Lesson 1, we explained the BOLD signal at one surface vertex. In Lesson 2, we stack every vertex side-by-side and fit the same design matrix to all vertices.

0. What we already know from Lesson 1

At one vertex, we have 128 measured BOLD values. The GLM tries to explain that one time series using the 18 design-matrix columns.

\[ \underbrace{y}_{128\times 1}=\underbrace{X}_{128\times 18}\underbrace{\beta}_{18\times 1}+\underbrace{\varepsilon}_{128\times 1} \]

1. Zero-abstraction picture: one vertex is one column of data

Imagine the surface has many vertices. At each vertex, the scanner gives a 128-number time series.

2. Stack many \(y\)'s side-by-side

For one vertex, the data vector is \(y_1\). For another vertex, it is \(y_2\). For another, \(y_3\). Put them next to each other:

\[ Y=\begin{bmatrix}|&|&|&&|\\ y_1&y_2&y_3&\cdots&y_N\\ |&|&|&&|\end{bmatrix} \]
\(Y\)
128 rows
\(N\) vertex columns

So \(Y\) is not one brain signal. It is a big table containing the BOLD signal from every vertex.

3. The design matrix \(X\) is reused at every vertex

The experiment timing is the same everywhere in the brain. So the design matrix is the same for vertex 1, vertex 2, vertex 3, and all other vertices.

Selected columns of the same design matrix \(X\)
The design matrix does not know where in the brain we are. It only knows when the tasks happened.

4. What changes across vertices? The coefficients

At each vertex, the GLM estimates 18 coefficients. These coefficients say how strongly each design column appears in that vertex's BOLD signal.

\[ B=\begin{bmatrix}|&|&|&&|\\ \beta_1&\beta_2&\beta_3&\cdots&\beta_N\\ |&|&|&&|\end{bmatrix}\in\mathbb{R}^{18\times N} \]

5. The full model

Now we can write the all-vertices model:

\[ \underbrace{Y}_{128\times N}=\underbrace{X}_{128\times 18}\underbrace{B}_{18\times N}+\underbrace{E}_{128\times N} \]

Matrix multiplication works because the inner dimensions match: \(18\) in \(X\) matches \(18\) in \(B\).

6. What does one row of \(B\) mean?

A row of \(B\) corresponds to one regressor across all vertices.

\[ \text{row 1 of }B=\text{condition 1 effect at every vertex} \]
Toy coefficient map: one row of \(B\) displayed across vertices
This is how coefficient maps appear: one beta value per vertex. Later, contrasts combine rows of \(B\).

8. Final memory hook

One-vertex model
\(y=X\beta+\varepsilon\)
One vertex, one coefficient vector.
All-vertices model
\(Y=XB+E\)
Many vertices, many coefficient vectors stacked together.

Plain English: we use the same 18 predicted time courses to explain every vertex, but each vertex gets its own set of 18 beta weights.
