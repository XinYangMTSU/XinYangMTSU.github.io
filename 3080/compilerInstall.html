Download & Install C++ Compiler

Windows

Install MinGW (Minimalist GNU for Windows). MinGW is a minimalist development environment for native Windows applications, which includes the GCC (GNU Compiler Collection) compiler, including g++ for C++. Steps to Install MinGW:
  • ★ Download the MinGW Installer:
    • 1. Go to the MinGW SourceForge page
    2. Click Download to get the MinGW Installation Manager.
  • ★ Run the Installer:
    • 1. Open the downloaded mingw-get-setup.exe file.
    2. Follow the prompts to install the MinGW Installation Manager.
  • ★ Select Packages:
    • 1. Open the MinGW Installation Manager.
    2. Select the following packages for installation:
    • mingw32-gcc-g++ (C++ Compiler)

    3. Right-click on each package and select Mark for Installation.
  • ★ Apply Changes:
    • 1. Click Installation in the top menu and then select Apply Changes.
    2. The installer will download and install the selected packages.
  • ★ Add MinGW to the System PATH:
    • 1. Open Control Panel > System and Security > System > Advanced System Settings.
    2. Click on Environment Variables.
    3. Under System variables, find Path, click Edit, and add the path to the MinGW bin directory (typically C:\MinGW\bin).
    4. Click OK to save.
  • ★ Verify Installation:
    • 1. Open Command Prompt and type: g++ --version
    2. If MinGW is installed correctly, it will display the version of g++.
