Bit

A bit is a binary digit, the smallest unit of data stored on a computer. A bit can hold only one of two values: 0 or 1, corresponding to the electrical values of off or on, respectively. Because bits are so small, you rarely work with information one bit at a time. Bits are usually assembled into a group of eight to form a byte.

Byte

A byte contains enough information to store a single ASCII character, like "A".
  • 1 byte = 8 bits
  • e.g.: 0 1 0 1 1 0 1 0
  • One byte can store one character, e.g. 'A' or 'x' or '$'
