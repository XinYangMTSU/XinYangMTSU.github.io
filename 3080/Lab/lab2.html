CSCI 3080 - Discrete Structures

Lab 2 - Virus Spread Simulation Using Recursion and Recurrence Relations

1. Objective

The goal of this lab is to develop a C++ program to simulate a virus spread model using recursion and recurrence relations.
Students will analyze the growth of an infection and compare recursive and iterative solutions.

2. Problem Statement

A virus spreads in a community based on the following pattern:

  • On Day 1, there is 1 infected person.
  • On Day 2, 2 more people are infected (total: 3).
  • On each day after that, the number of new infections follows the recurrence:

Recurrence Relation:

\[ S(n) = \begin{cases} 1, & \text{if } n = 1 \\ 2, & \text{if } n = 2 \\ S(n-1) + 2\timesS(n-2), & \text{if } n \geq 3 \end{cases} \]

3. Requirements

  • Implement a recursive function to compute \( I(n) \).
  • Implement an iterative function to compute \( I(n) \).
  • Compare the results of recursion and iteration.

4. Sample Input/Output

Enter the number of days (n >= 1): 6
Recursive Infection Model: I(6) = 63
Iterative Infection Model: I(6) = 63

5. Grading Criteria

Iterative Function Implementation 10
