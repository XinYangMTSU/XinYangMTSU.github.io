Lab 5 - Compressing Text Files using Huffman Coding

1. Objective

In this lab, students will implement Dijkstra's algorithm in C++ to find the shortest path in a weighted graph. They will learn how to represent graphs using adjacency matrices and vectors, trace the algorithm step by step, and reconstruct the shortest path from the start node to the destination. Additionally, students will visualize how the algorithm processes each node and updates distances in a priority-based manner. By the end of this lab, students will be able to:

  • ★ Understand the concept and step-by-step logic behind Dijkstra's algorithm.
  • ★ Represent graphs using adjacency matrices and vectors in C++.
  • ★ Implement Dijkstra’s algorithm to compute shortest distances from a given source node.
  • ★ Reconstruct and display the shortest path from source to destination.
  • ★ Analyze how the algorithm chooses the next node and updates its neighbors.
  • ★ Visualize and explain Dijkstra’s steps using tables or animations.

  • ★ (3) Please use a vector to construct the Huffman tree by merging nodes with the lowest frequencies and store the root node of the tree.
  • ★ (4) Please traverse the Huffman tree to assign binary codes to each character and store the codes in an unordered_map.
