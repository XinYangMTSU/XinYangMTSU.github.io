CSCI 3080 - Discrete Structures

Bonus Lab - Binary, Decimal, Hexadecimal Converter

1. Objective

The goal of this lab is to create a C++ program that converts a number between three numbering systems: binary, decimal, and hexadecimal. Students will gain hands-on experience in base conversions and string manipulation.

2. Requirements

  • ★ (1) Ask the user to choose the input number system (Binary, Decimal, or Hexadecimal).
  • ★ (2) Prompt the user to enter a number in that base.
  • ★ (3) Convert and display the number in all three formats.
  • ★ (4) Use input validation for each base.
  • ★ (5) Include a do-while loop to allow repeat usage.

3. Sample Input/Output

Number System Converter
---------------------------
Choose input format:
1. Binary
2. Decimal
3. Hexadecimal
Enter your choice (1-3): 2
Enter a decimal number: 255

Results:
Binary: 11111111
Decimal: 255
Hexadecimal: FF

Do you want to convert another number? (y/n): y
