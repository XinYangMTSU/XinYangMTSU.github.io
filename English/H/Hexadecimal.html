🧮 hexadecimal /ˌhɛksəˈdɛsɪməl/

Part of speech: Adjective / Noun
词性：形容词 / 名词

📖 Meaning:

Relating to a base-16 number system that uses digits 0–9 and letters A–F to represent values from 0 to 15. Commonly used in computing and digital systems.
十六进制的；一种以 16 为底的数字系统，使用 0–9 和 A–F 表示 0 到 15 的值，广泛应用于计算机与数字系统。

Example: The color code #FF5733 is written in hexadecimal format.
例句：颜色代码 #FF5733 是用十六进制表示的。
Example: In programming, hexadecimal values are often prefixed with 0x.
例句：在编程中，十六进制数通常以前缀 0x 表示。

🧩 Synonyms /ˈsɪnəˌnɪmz/

  • Base-16 十六进制
  • Hex 十六进制（口语）
  • Numeral system 数字系统
  • Hex code 十六进制代码

🔍 Etymology of the word /ˌɛtəˈmɑːlədʒi/

📚 "Hexadecimal" comes from the prefix hexa- (Greek for "six") + decimal (Latin decem = "ten"), meaning "six and ten" or base-16.
“Hexadecimal” 源自希腊语前缀 hexa-（六）和拉丁语 decimal（十），组合表示“六加十”即“以 16 为底”。

🧭 Word Evolution

Greek: hexa – six
希腊语 hexa：六

Latin: decem – ten
拉丁语 decem：十

Modern English: hexadecimal – base-16 numeral system
现代英语 hexadecimal：十六进制数字系统
