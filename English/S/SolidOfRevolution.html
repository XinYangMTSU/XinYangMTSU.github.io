🔄 solid of revolution /ˈsɒlɪd əv ˌrɛvəˈluːʃən/

Part of speech: Noun phrase
词性：名词短语

📖 Meaning:

A solid of revolution is a 3D shape formed by rotating a 2D curve or region around an axis. The rotation creates a symmetrical solid, often with circular cross-sections.
solid of revolution（旋转体）是通过将一个二维图形绕某条轴旋转所形成的三维图形。旋转产生的物体具有对称性，截面通常是圆形。

Example: Rotating the curve y = x² from x = 0 to x = 1 around the x-axis forms a solid of revolution.
例句：将 y = x² 从 x = 0 到 x = 1 的曲线绕 x 轴旋转可形成一个旋转体。
Example: Engineers use solids of revolution to model objects like nozzles, vases, and machine parts.
例句：工程师利用旋转体来建模喷嘴、花瓶和机器部件等物体。

🔍 Concept Origin

📚 The term "solid of revolution" comes from geometry and calculus. It refers to the **revolution (rotation)** of a 2D shape around an axis to create a **solid (3D) object**.
“solid of revolution” 源自几何学和微积分，表示将一个二维图形绕某条轴旋转，生成一个三维物体。

🧭 Formation Example

2D curve: y = x² from x = 0 to x = 1
二维曲线：y = x²，范围是 x = 0 到 x = 1
🔄
Rotated around: x-axis
绕 x 轴旋转
📐
Result: A paraboloid-shaped solid
结果：一个抛物旋转体
