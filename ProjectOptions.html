Final Project Options

1. Web-based Lab System

  • ★ The students can at least do multiple choice questions online.
  • ★ The user can check student's grades and questions information.

2. Web-based Food Order Management System

  • ★ The user can search for the food options online.
  • ★ The user can also order the food online.

3. Web-based Library Management System

  • ★ The user can search the information and availability of the books online.
  • ★ The user can also reserve the book if the book is available.

4. Web-based Planner

  • ★ The user can enter and save all the schedule online.
  • ★ The user can also search for the specific dates' schedule.

5. Web-based Online Survey System

  • ★ The user can enter the answers for all the questions.
  • ★ The user can also see the statistic results for the survey.
