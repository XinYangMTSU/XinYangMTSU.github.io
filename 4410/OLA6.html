OLA 6: HTML, CSS, PHP, and MySQL — Student Database Project

Course: CSCI 4410/5410 Web Technologies

Instructor: Dr. Xin Yang

Due date: 11:59 pm TBA

Objective:

In this assignment, you will build a dynamic PHP application connected to a MySQL database to manage student records. You will create a database, design a table, perform SQL queries, and format the results with HTML and CSS.

Step 1: Create Your Database and Table (30 Points)

  • Create a database named CSCI4410 in phpMyAdmin.
  • Create a table named Students with fields: ID (Auto_Increment Primary Key), Name, BlueCard, Major, ClassLevel, Email, Gender, Age, Phone.
  • Insert 10 student records into your Students table (see assignment page for detailed data).
  • Save your SQL file as Students.sql.

Step 2: Create Your PHP File (60 Points)

  • Create a PHP file named index.php.
  • Connect to the database using object-oriented PHP: new mysqli().
  • Perform the following queries and display results in an HTML table:
    • Display all students.
    • Display only male students.
    • Display only female students.
    • Display students older than 21.
    • Display the number of distinct majors.
    • Display students without phone numbers (where phone IS NULL).
    • Insert a new record and confirm insertion.
    • Delete the new record and confirm deletion.
    • Update John Doe’s phone number and confirm update.
  • Close the database connection at the end.

Step 3: Create Your CSS File (10 Points)

  • Create a CSS file named ola6.css.
  • Style your webpage and table for readability and good appearance.
  • Link your CSS in your PHP file using:
    <link rel="stylesheet" href="ola6.css">

Step 4: Submission Instructions

Submit the following files to edStem:

  • Students.sql
  • index.php
  • ola6.css

Important: Test everything locally using XAMPP before submitting!

Grading:

Part Points
Database and SQL (Students.sql) 30 Points
PHP Code and Queries (index.php) 60 Points
CSS Design (ola6.css) 10 Points
