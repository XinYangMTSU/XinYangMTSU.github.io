React + PHP + MySQL JWT Authentication Project

The React + PHP + MySQL JWT Authentication project is a full-stack user authentication system that enables secure user registration, login, and session management using JWT (JSON Web Token). The frontend, built with React, provides a seamless user interface for the account creation and authentication, sending API requests to a PHP-based backend. The backend, powered by PHP and MySQL, handles user data storage, password hashing for security, and JWT generation for authentication. Users register with a username and password, which are securely stored in a MySQL database. Upon login, a JWT token is issued, allowing authenticated users to access protected routes. The system ensures security with password hashing (bcrypt) and JWT-based session management, making it a robust and scalable authentication solution.

1. Set Up MySQL Database

Create Database & Users Table

  • (1) Open phpMyAdmin (http://localhost/phpmyadmin/).
  • (2) Run the following SQL commands to create the database and users table: 
    • 				
				CREATE DATABASE user_auth;
				USE user_auth;
				
				CREATE TABLE users (
				    id INT AUTO_INCREMENT PRIMARY KEY,
				    username VARCHAR(100) UNIQUE NOT NULL,
				    password VARCHAR(255) NOT NULL
				);

2. Set Up PHP Backend (XAMPP)

A. Create Project Folder

  • (1) Inside htdocs (XAMPP folder), create a new project:
    • 				
					cd /Applications/XAMPP/xamppfiles/htdocs
					mkdir react-php-auth
					cd react-php-auth
  • (2) Inside react-php-auth, create these PHP files:
    • 				db.php
				register.php
				login.php
				jwt_helper.php

Step 1: Check If PHP Is Installed in XAMPP

Run: 
			/Applications/XAMPP/xamppfiles/bin/php -v
If PHP is installed, you should see something like: 
			PHP 8.x.x (cli) (built: ...)

Step 2: Add XAMPP PHP to Your System PATH

Since you're using XAMPP, the PHP binary is located in: 
			/Applications/XAMPP/xamppfiles/bin/php
To make it accessible globally, run: 
			echo 'export PATH="/Applications/XAMPP/xamppfiles/bin:$PATH"' >> ~/.zshrc
			source ~/.zshrc
For Bash Users: 
			echo 'export PATH="/Applications/XAMPP/xamppfiles/bin:$PATH"' >> ~/.bashrc
			source ~/.bashrc
Now, check if PHP is working: 
			php -v

Step 3: Install Composer

Once PHP is working, install Composer: 
			curl -sS https://getcomposer.org/installer | PHP
			sudo mv composer.phar /usr/local/bin/composer
Verify Composer installation: 
			composer -v

Step 4: Install JWT Library

Navigate to your PHP project folder (react-php-auth) and run: 
			cd /Applications/XAMPP/xamppfiles/htdocs/react-php-auth
			composer require firebase/php-jwt
