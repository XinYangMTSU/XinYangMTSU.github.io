Introduction to React Syntax

1. JSX (JavaScript XML)

JSX allows writing HTML-like syntax inside JavaScript. It makes React code easier to read and write.

Example without JSX (Regular JavaScript)

const element = React.createElement("h1", null, "Hello, React!");
        import React from "react";

        function App() {
          // Creating an element using React.createElement
          const element = React.createElement("h1", null, "Hello, React!");

          return element;
        }

        export default App;

This is how you would write React elements without JSX.

Example with JSX

const element = <h1>Hello, React!</h1>;

This is the same as the previous example but using JSX.

Key JSX Rules:

Only one parent element:

return (
  <div>
    <h1>Title</h1>
    <p>Paragraph</p>
  </div>
);

Or, use fragments:

return (
  <>
    <h1>Title</h1>
    <p>Paragraph</p>
  </>
);

Expressions inside JSX:

const name = "Alice";
const element = <h1>Hello, {name}!</h1>;

Attributes in JSX:

const imgUrl = "https://example.com/image.jpg";
const element = <img src={imgUrl} alt="Example Image" />;

2. Components

React applications are built using components. A component is a function that returns JSX.

Basic Functional Component

function Greeting() {
  return <h1>Hello, React!</h1>;
}

export default Greeting;

Using the Component

In App.js:

import Greeting from "./Greeting";

function App() {
  return (
    <div>
      <Greeting />
    </div>
  );
}

export default App;

3. Rendering a Component

React renders components inside index.js using ReactDOM.createRoot().

Example (index.js)

import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

const root = ReactDOM.createRoot(document.getElementById("root"));
root.render(<App />);

4. Props (Passing Data to Components)

Props (short for properties) allow data to be passed to components.

Example:

function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

Passing Props to a Component:

<Welcome name="Alice" />

5. State (Managing Component Data)

Example using useState:

import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increase</button>
    </div>
  );
}

export default Counter;

6. Handling Events

Example:

function ClickHandler() {
  function handleClick() {
    alert("Button clicked!");
  }

  return <button onClick={handleClick}>Click Me</button>;
}

export default ClickHandler;

7. Conditional Rendering

Example:

function Message(props) {
  return props.isLoggedIn ? <h1>Welcome Back!</h1> : <h1>Please Log In</h1>;
}

8. Lists and Keys

Example:

const names = ["Alice", "Bob", "Charlie"];

function NameList() {
  return (
    <ul>
      {names.map((name, index) => (
        <li key={index}>{name}</li>
      ))}
    </ul>
  );
}

9. Forms and Input Handling

Example:

import { useState } from "react";

function Form() {
  const [text, setText] = useState("");

  function handleChange(event) {
    setText(event.target.value);
  }

  return (
    <div>
      <input type="text" value={text} onChange={handleChange} />
      <p>You typed: {text}</p>
    </div>
  );
}

10. useEffect (Handling Side Effects)

Example:

import { useState, useEffect } from "react";

function Timer() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    console.log("Component updated! Count:", count);
  }, [count]);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increase</button>
    </div>
  );
}
