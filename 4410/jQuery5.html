jQuery Effects - KeyUp & KeyDown

  • ★ The keyup() method triggers the keyup event, or attaches a function to run when a keyup event occurs.
  • ★ The keydown() method triggers the keydown event, or attaches a function to run when a keydown event occurs.
			In JavaScript:
			
			$(document).ready(function(){
  				$("input").keydown(function(){
    					$("input").css("background-color", "yellow");
  				});
 					
				$("input").keyup(function(){
    					$("input").css("background-color", "pink");
 				 });
			});

			In HTML:

			Enter your name: <input type="text">
			<p>Enter your name in the input field above. It will change background color on keydown and keyup.</p>

jQuery Effects - Animation

With jQuery, you can create custom animations.

The animate() Method

Syntax: 
			$(selector).animate({params},speed,callback);
  • ★ The required params parameter defines the CSS properties to be animated.
  • ★ The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast", or milliseconds.
  • ★ The optional callback parameter is a function to be executed after the animation completes.
  • ★ The following example demonstrates a simple use of the animate() method; it moves a
    element to the right, until it has reached a left property of 250px:
    • 				In JavaScript:
				$(document).ready(function(){
  				$("button").click(function(){
    				$("div").animate({left: '250px'});
  				});
				});
				In HTML:

				<button>Start Animation</button>

				<p>By default, all HTML elements have a static position, and cannot be moved. To manipulate the position, remember to first set the CSS position property of the element to relative, fixed, or absolute!<<p>

				<div style="background:#98bf21;height:100px;width:100px;position:absolute;"></div>

jQuery animate() - Manipulate Multiple Properties

Notice that multiple properties can be animated at the same time: 
			In JavaScript:
			$(document).ready(function(){
  				$("button").click(function(){
    					$("div").animate({
      					left: '250px',
      					opacity: '0.5',
      					height: '150px',
      					width: '150px'
    					});
  				});
			});

			In HTML:
			<button>Start Animation</button>

			<p>By default, all HTML elements have a static position, and cannot be moved. To manipulate the 				position, remember to first set the CSS position property of the element to relative, fixed, or 				absolute!</p>

			<div style="background:#98bf21;height:100px;width:100px;position:absolute;"></div>

jQuery animate() - Using Relative Values

  • ★ It is also possible to define relative values (the value is then relative to the element's current value).
  • ★ This is done by putting += or -= in front of the value:
    • 				 In JavaScript:
				$(document).ready(function(){
				  $("button").click(function(){
    					$("div").animate({
      					 left: '+=50px',
     					height: '+=150px',
      					 width: '+=150px'
    					});
  				});
				});

				In HTML:
				<button>Start Animation</button>

				<p>By default, all HTML elements have a static position, and cannot be moved. 
				To manipulate the position, remember to first set the CSS position property of the 
				element to relative, fixed, or absolute!</p>

				<div style="background:#98bf21;height:100px;width:100px;position:absolute;"></div>
