OLA 4: React Blog Website

Course: CSCI 4410/5410 Web Technologies

Instructor: Dr. Xin Yang

Due date: 11:59 pm Mar 25th, 2025 (Tuesday)

Objective:

In this assignment, you will develop a dynamic personal blog application using React. Your task is to create an interactive interface where users can add, view, and delete blog posts while navigating between different pages using React Router. You will implement state management to handle blog data, use event handling for form submissions, and structure your application using reusable components. By completing this project, you will enhance your ability to build and manage a multi-page React application while applying best practices in component organization, routing, and user input handling.

Step 1: Install Dependencies

First, initialize a React app and install react-router-dom: 
		npx create-react-app my-react-blog
		cd my-react-blog
		npm install react-router-dom

Project Structure

	/my-react-blog
	│── /src
	│   │── /components
	│   │   ├── Navbar.js
	│   │── /pages
	│   │   ├── Home.js
	│   │   ├── Blog.js
	│   │   ├── NewPost.js
	│   │   ├── PostDetail.js
	│   │── App.js
	│   │── index.js
	│   │── styles.css

Step 2: React Component Development (30 Points)

(1) ♣ Create React components for different pages (Home.js, Blog.js, NewPost.js, PostDetail.js).
(2) ♣ Implement state management (useState) for handling blog posts dynamically.
(3) ♣ Ensure each component is well-structured and functional.

Step 3: Implementing React Router (20 Points)

(1) ♣ Install and configure React Router for multi-page navigation.
(2) ♣ Use the <Link> component for navigation.
(3) ♣ Handle 404 (Not Found) pages correctly.

Step 4: Form Handling & Event Management (20 Points)

(1) Implement a form in NewPost.js to allow users to add blog posts.
(2) Use controlled components (useState) for form input management.
(3) Ensure form validation (e.g., title, author, and content fields cannot be empty).
(4) After submission, update the blog post list dynamically.

Step 5: Post Viewing & Deletion (20 Points)

(1) Create PostDetail.js to display full blog post details.
(2) Implement Delete functionality to remove posts from the list.
(3) Use React state updates to dynamically reflect deletions.

Step 6: Styling & UI Enhancement (10 Points)

(1) Apply CSS to enhance the appearance of the blog.
(2) Style navigation links, form inputs, buttons, and blog post lists.
(3) Ensure a responsive layout for better user experience.

Step 7: Submission Instructions

Please compress your folder my-react-blog into a single ZIP archive and submit it through D2L.
