jQuery - Get and Set Content and Attributes

  • ★ One very important part of jQuery is the possibility to manipulate the DOM.
  • ★ jQuery comes with a bunch of DOM-related methods that make it easy to access and manipulate elements and attributes.
  • ★ DOM = Document Object Model
  • ★ The DOM defines a standard for accessing HTML and XML documents.

Get Content - text(), html(), and val()

Three simple, but useful, jQuery methods for DOM manipulation are:
  • ★ text() - Sets or returns the text content of selected elements.
  • ★ html() - Sets or returns the content of selected elements (including HTML markup).
  • ★ val() - Sets or returns the value of form fields.
The following example demonstrates how to get content with the jQuery text() and html() methods: 
		In JavaScript:

		$(document).ready(function(){
  			$("#btn1").click(function(){
    				alert("Text: " + $("#test").text());
  			});
  			$("#btn2").click(function(){
   				 alert("HTML: " + $("#test").html());
  			});
		});

		In HTML:

		<p id="test">This is some <b>bold</b> text in a paragraph.</p>

		<button id="btn1">Show Text</button>
		<button id="btn2">Show HTML</button>
The following example demonstrates how to get the value of an input field with the jQuery val() method: 

				In JavaScript:
				
				$(document).ready(function(){
 					 $("button").click(function(){
    					alert("Value: " + $("#test").val());
  				});
				});

				In HTML:
				
				<p>Name: <input type="text" id="test" value="Mickey Mouse"></p>
				<button>Show Value</button>

Get Attributes - attr()

  • ★ The jQuery attr() method is used to get attribute values.
  • ★ The following example demonstrates how to get the value of the href attribute in a link:
			In JavaScript:

			$(document).ready(function(){
  				$("button").click(function(){
    					alert($("#w3s").attr("href"));
  				});
			});

			In HTML:

			<p><a href="https://www.w3schools.com" id="w3s">W3Schools.com</a></p>
			<button>Show href Value</button>

Set Content - text(), html(), and val()

The following example demonstrates how to set content with the jQuery text() , html() and vall() methods 
				In JavaScript:

				$(document).ready(function(){
  					$("#btn1").click(function(){
    					$("#test1").text("Hello world!");
  					});
  					$("#btn2").click(function(){
    					$("#test2").html("<b>Hello world!</b>");
  					});
  					$("#btn3").click(function(){
    					$("#test3").val("Dolly Duck");
  					});
				});

				In HTML:

				<p id="test1">This is a paragraph.</p>
				<p id="test2">This is another paragraph.</p>

				<p>Input field: <input type="text" id="test3" value="Mickey Mouse"></p>

				<button id="btn1">Set Text</button>
				<button id="btn2">Set HTML</button>
				<button id="btn3">Set Value</button>

Set Attributes - attr()

  • ★ The jQuery attr() method is also used to set/change attribute values.
  • ★ The following example demonstrates how to change (set) the value of the href attribute in a link:
    • 				In JavaScript:
				
				$(document).ready(function(){
  					$("button").click(function(){
   						 $("#w3s").attr("href", "https://www.w3schools.com/jquery/");
    					 	$("a").text("jQuery Tutorial");
  					});
				});

				In HTML:
				
				<p><a href="https://www.w3schools.com" id="w3s">W3Schools.com</a></p>

				<button>Change href Value</button>
