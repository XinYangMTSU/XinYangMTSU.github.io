OLA 1: HTML (Industry Workflow)

Course: CSCI 4410/5410 Web Technologies

Instructor: Dr. Xin Yang

Due date: 11:59 pm Feb TBD, 2026 ()

Objective:

In this assignment, you will create a personal webpage using an industry-inspired workflow: (1) plan your structure, (2) build your HTML, and (3) reflect on what changed and why. The goal is to showcase your understanding of basic HTML components, including <!DOCTYPE>, <html>, <head>, <body>, <meta>, <title>, <p>, <img>, <a>, <ul>, <ol>, <li>, <table>, <tr>, <th>, <td>, <thead>, <tbody>, <b> or <strong>, <i> or <em>, <mark>, and HTML comments <!-- -->.

Rules:

  • • HTML only (we have not taught CSS or JavaScript yet).
  • • Do NOT use CSS: no <style>, no style="", no external CSS files.
  • • Do NOT use JavaScript: no <script>.
  • • Avoid deprecated tags such as <font> and <center> in your student submission.
  • • Your HTML file must be named ola1.html.

Industry Workflow Steps (3 Stages)

You must submit all three stages:
  1. Stage 1: Hand-drawn or text outline of HTML structure
  2. Stage 2: HTML implementation (personal webpage)
  3. Stage 3: Reflection: what changed and why

Stage 1: Plan (Outline of Structure) (30 points)

Before writing code, plan the structure of your personal webpage (this is how web developers work in industry).

What to submit

  • • Option A: A clear photo of a hand-drawn outline, OR
  • • Option B: A text file named outline.txt

Outline requirements

Your outline must show the order of your webpage and include:
  • • A top title area (H1)
  • • At least 4 content sections (each will later have a heading)
  • • One section that contains a table
  • • A bottom area (closing content + reflection section)

Example outline (text)

Top
- H1: My Personal Homepage

Section 1: About Me
- 1 paragraph with formatting

Section 2: Image
- Image + alt + source/credit (if needed) + short explanation

Section 3: Lists
- UL + OL + nested list

Section 4: Favorites Table
- thead + tbody + 3x3 data + rowspan/colspan

Section 5: External Link
- link to a meaningful website

Section 6: Reflection
- what changed and why
Important: Your Stage 2 HTML should closely follow your Stage 1 outline. If you change the structure, explain it in Stage 3.

Stage 2: Build (HTML Implementation) (50 points)

Create your personal webpage in a file named ola1.html.

Basic HTML structure

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title></title>
</head>
<body>

</body>
</html>

Stage 2 requirements (personal webpage)

1. Title

Add a title in the head section using <title>. The title should match your page’s main heading.

2. About Me (Introduction Paragraph)

Write one paragraph to introduce yourself (name, where you're from, academic interests, hobbies/goals). This paragraph must include:
  • • one <strong> or <b>
  • • one <em> or <i>
  • • one <mark>
  • • exactly one <br>

3. Image (with alt + credit)

Include one image using <img>.
  • • You must include an alt attribute.
  • • If the image is from the internet, include a visible credit link to the source using <a>.
  • • Under the image, write 1–2 sentences explaining why you chose it.

4. Lists (unordered, ordered, nested)

Create:
  • • One <ul> with at least 4 items
  • • One <ol> with at least 3 items
  • • At least one nested list (a list inside an <li>)

5. Table (use thead + tbody)

Create a table listing your favorites (movies/books/songs/games/etc.). Requirements:
  • • Use <thead> and <tbody>.
  • • Include at least 3 rows of data in <tbody>.
  • • Include at least 3 columns.
  • • Use rowspan or colspan at least once.

6. External Hyperlink

Add one external hyperlink to a meaningful website (favorite website, club, hobby, portfolio inspiration, etc.). The link must open in a new tab using target="_blank".

7. Headings

Use headings (e.g., <h1>, <h2>, <h3>) to separate each section of your webpage.

8. Comments (exactly 6)

Add exactly 6 HTML comments that explain key parts of your page structure. Example: 
<!-- This section introduces who I am -->

Stage 3: Review (Reflection) (20 points)

At the bottom of your webpage, add a section titled Reflection. Answer each question in 2–3 sentences:
  1. What changed between your Stage 1 outline and your final HTML? Why did you change it?
  2. What was the hardest part to implement correctly, and why?
  3. Describe one mistake you made and how you fixed it.
  4. If you had more time (still HTML-only), what would you improve next?
Anti-copy-paste note: Your reflection must match your actual HTML file. Reflections that do not match the code will lose points.

  • • If you used a local image, submit the image file too.
