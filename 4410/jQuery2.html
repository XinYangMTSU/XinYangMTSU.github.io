jQuery Events Hide and Show

jQuery hide() and show()

With jQuery, you can hide and show HTML elements with the hide() and show() methods: 
			In JavaScript:
			
			$(document).ready(function(){
  				$("#hide").click(function(){
    					$("p").hide();
  				});
			
  				$("#show").click(function(){
    					$("p").show();
  				});
			});

			In HTML:

			<p>If you click on the "Hide" button, I will disappear.</p>

			<button id="hide">Hide</button>
			<button id="show">Show</button>

Syntax:

		$(selector).hide(speed,callback); 
		$(selector).show(speed,callback);
  • ★ The optional speed parameter specifies the speed of the hiding/showing, and can take the following values: "slow", "fast", or milliseconds.
  • ★ The optional callback parameter is a function to be executed after the hide() or show() method completes
  • ★ The following example demonstrates the speed parameter with hide():
    • 				$("button").click(function(){
  					$("p").hide(1000);
				});

jQuery toggle()

  • ★ You can also toggle between hiding and showing an element with the toggle() method.
  • ★ Shown elements are hidden and hidden elements are shown:
    • 				In JavaScript:
				
				$(document).ready(function(){
  					$("button").click(function(){
    					$("p").toggle();
  					});
				});

				In HTML:
				<button>Toggle between hiding and showing the paragraphs</button>

				<p>This is a paragraph with little content.</p>
				<p>This is another small paragraph.</p>

Syntax:

		$(selector).toggle(speed,callback);

jQuery Effects - Fading

  • ★ With jQuery you can fade an element in and out of visibility.
  • ★ jQuery has the following fade methods:
    • ☆ fadeIn()
    • ☆ fadeOut()
    • ☆ fadeToggle()
    • ☆ fadeTo()

jQuery fadeIn()

Syntax:

		$(selector).fadeIn(speed,callback);
  • ★ The jQuery fadeIn() method is used to fade in a hidden element.
  • ★ The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast", or milliseconds.
  • ★ The optional callback parameter is a function to be executed after the fading completes.
  • ★ The following example demonstrates the fadeIn() method with different parameters:
    • 				In JavaScript:
				
				$(document).ready(function(){
  					$("button").click(function(){
    						$("#div1").fadeIn();
    						$("#div2").fadeIn("slow");
    						$("#div3").fadeIn(3000);
  					});
				});

				In HTML:
				<button>Click to fade in boxes</button><br><br>

				<div id="div1" style="width:80px;height:80px;display:none;background-color:red;"></div><br>
				<div id="div2" style="width:80px;height:80px;display:none;background-color:green;"></div><br>
				<div id="div3" style="width:80px;height:80px;display:none;background-color:blue;"></div>

jQuery fadeOut()

Syntax:

		$(selector).fadeOut(speed,callback);
  • ★ The jQuery fadeOut() method is used to fade out a visible element.
  • ★ The optional speed parameter specifies the duration of the effect. It can take the following values: "slow", "fast", or milliseconds.
  • ★ The optional callback parameter is a function to be executed after the fading completes.
  • ★ The following example demonstrates the fadeOut() method with different parameters:
    • 				In JavaScript:
				
				$(document).ready(function(){
  					$("button").click(function(){
    					  	$("#div1").fadeOut();
    						$("#div2").fadeOut("slow");
    						$("#div3").fadeOut(3000);
  					});
				});

				In HTML:
				<button>Click to fadeOut boxes</button><br><br>

				<div id="div1" style="width:80px;height:80px;background-color:red;"></div><br>
				<div id="div2" style="width:80px;height:80px;background-color:green;"></div><br>
				<div id="div3" style="width:80px;height:80px;background-color:blue;"></div>

jQuery fadeToggle()

Syntax:

		$(selector).fadeToggle(speed,callback);
  • ★ The jQuery fadeToggle() method toggles between the fadeIn() and fadeOut() methods.
  • ★ If the elements are faded out, fadeToggle() will fade them in.
  • ★ If the elements are faded in, fadeToggle() will fade them out.
  • ★ The following example demonstrates the fadeToggle() method with different parameters:
    • 				In JavaScript:
				
				$(document).ready(function(){
  					$("button").click(function(){
    					  	$("#div1").fadeToggle();
    						$("#div2").fadeToggle("slow");
    						$("#div3").fadeToggle(3000);
  					});
				});

				In HTML:
				<button>Click to fade in/out boxes</button><br><br>

				<div id="div1" style="width:80px;height:80px;background-color:red;"></div><br>
				<div id="div2" style="width:80px;height:80px;background-color:green;"></div><br>
				<div id="div3" style="width:80px;height:80px;background-color:blue;"></div>

jQuery fadeTo()

Syntax

			$(selector).fadeTo(speed,opacity,callback);
  • ★ The jQuery fadeTo() method allows fading to a given opacity (value between 0 and 1).
  • ★ fadeTo() takes three arguments, or inputs, between its parentheses, separated by a comma:
    • argument explanation values
    speed the duration of the effect "slow", "fast", or milliseconds.
    opacity fading to a given opacity value between 0 and 1
    callback a function to be executed after the function complete
  • ★ The following example demonstrates the fadeTo() method with different parameters:
    • 				In JavaScript:
				
				$(document).ready(function(){
  					$("button").click(function(){
    					$("#div1").fadeTo("slow", 0.15);
    					$("#div2").fadeTo("slow", 0.4);
    					$("#div3").fadeTo("slow", 0.7);
  					});
				});

				In HTML:
				<button>Click to fade boxes</button><br><br>

				<div id="div1" style="width:80px;height:80px;background-color:red;"></div><br>
				<div id="div2" style="width:80px;height:80px;background-color:green;"></div><br>
				<div id="div3" style="width:80px;height:80px;background-color:blue;"></div>

jQuery Effects - Sliding

jQuery slideToggle() Method

Syntax:

		$(selector).slideToggle(speed,callback);
  • ★ The jQuery slideToggle() method toggles between the slideDown() and slideUp() methods.
  • ★ If the elements have been slid down, slideToggle() will slide them up.
  • ★ If the elements have been slid up, slideToggle() will slide them down.
				In JavaScript:
				
				$(document).ready(function(){
  					$("#flip").click(function(){
    					$("#panel").slideToggle("slow");
  					});
				});

				In CSS:

				#panel, #flip {
  						padding: 5px;
  						text-align: center;
  						background-color: #e5eecc;
  						border: solid 1px #c3c3c3;
				}

				#panel {
  						padding: 50px;
  						display: none;
				}
				
				In HTML:

				<div id="flip">Click to slide the panel down or up</div>
				<div id="panel">Hello world!</div>
