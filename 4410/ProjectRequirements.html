Final Project Guidelines and Grading Criteria

Your final project needs to cover all the following topics:

  • ★ Front End : HTML/CSS, JavaScript/React
  • ★ Back End : Server/PHP, MySQL/Database
  • ★ Final Report (Maximum 5 pages, Minimum 2 pages)
    • ☆ Project Description
    • ☆ Project Design Diagram Graph
    • ☆ Team Member Task Distribution
  • ★ Final Presentation
    • ☆ Each Team has 10 minutes for demonstration.
  • ★ Your website should have at least one feature was not taught in the class.

Please Submit all your files in D2L:

  • ★ PHP/HTML file
  • ★ JavaScript file
  • ★ CSS file
  • ★ SQL file
  • ★ Final Report PDF file

Grading

There are 100 points in total. Here is the grading scale:

Part Points
Project Visual Design (HTML/CSS) 15 Points
Project Functionality (JavaScript/React) 15 Points
Project Server (PHP) 20 Points
Project Database (MySQL) 20 Points
New Feature 10 points
Project Presentation 10 Points
Project Report PDF 10 Points
